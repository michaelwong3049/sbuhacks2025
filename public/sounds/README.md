# Sound Samples

## Triangle Sample

To use a real triangle sample for better sound quality:

1. Download a triangle percussion sample (MP3 or WAV format)
2. Place it in this directory as `triangle.mp3` or `triangle.wav`
3. The Triangle class will automatically detect and use it

### Recommended Sources:
- **Freesound.org**: https://freesound.org (search for "triangle percussion")
- **Zapsplat**: https://www.zapsplat.com (free with attribution)
- **BBC Sound Effects**: https://sound-effects.bbcrewind.co.uk

### Sample Requirements:
- Format: MP3 or WAV
- Duration: 1-3 seconds (triangle ring)
- Quality: 44.1kHz sample rate recommended
- File name: `triangle.mp3` or `triangle.wav`

If no sample is found, the app will use synthesized triangle sound.
